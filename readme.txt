== Installation ==
Use WordPress' Add New Plugin feature, searching "Nomad Free Plugin", or download the archive and:
1. Unzip the archive on your computer
2. Upload `nomad-plugin` directory to the '/wp-content/plugins/' directory
3. Activate the plugin through the 'Plugins' menu in WordPress
4. New menu item should appear on the admin menu at the first level
